
# Movie Recommendation System

A content-based Movie Recommendation System built using **Python**, **Flask**, and **Machine Learning**.  
The application recommends similar movies based on user-selected movie preferences.

---

# Live Demo

🔗 https://movie-recommendation-system-wwvf.onrender.com

---

#  Features

- Recommend similar movies instantly
- Search movies by title
- Fast recommendation system using cosine similarity
- Fully deployed on Render
- Interactive frontend using HTML, CSS, and JavaScript
- Machine Learning powered recommendation engine

# Installation & Setup

## Clone Repository

```bash
git clone https://github.com/priyanshu-iitp/Movie-recommendation-system.git
```

---

## Move to Project Directory

```bash
cd Movie-recommendation-system/backend
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

## Run Application

```bash
python app.py
```

---

# How It Works

1. Movie datasets are processed using NLP techniques.
2. Feature vectors are created from movie metadata.
3. Cosine similarity calculates similarity scores between movies.
4. Flask backend serves recommendations dynamically.

---

# Dataset Used

* TMDB 5000 Movies Dataset
* TMDB 5000 Credits Dataset
